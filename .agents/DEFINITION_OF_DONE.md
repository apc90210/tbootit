# DEFINITION OF DONE — Техноребут MVP

## Для любого модуля

Модуль готов только если:

- код реализован;
- Docker запускается;
- API отвечает;
- основные сценарии проверены;
- документация обновлена;
- отчет создан;
- git commit создан;
- ограничения честно указаны.

## Для Core MVP

Core MVP готов, если:

```text
docker compose up --build работает
http://127.0.0.1:8000/health отвечает
http://127.0.0.1:8000/docs открывается
http://127.0.0.1:8010 открывается
можно создать товар
можно посмотреть товар
можно изменить статус товара
можно загрузить фото
можно создать клиента
можно создать ремонт
можно провести продажу
можно посмотреть структуру БД
можно посмотреть audit log
данные сохраняются после перезапуска
```

## Для перехода к внешним модулям

Переход к отдельным внешним модулям разрешен только после:

```text
Core MVP принят
API-контракт стабилен на базовом уровне
ручное тестирование пройдено или замечания зафиксированы
```
