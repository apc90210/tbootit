# Stage 04G Sales Reports and Payment Channels

## Описание
В этом этапе был реализован новый раздел "Отчёты по продажам" и добавлены новые каналы поступления денег, в том числе "Счёт юрлица" и "СБП". Отчёт предоставляет агрегированную финансовую статистику за выбранные периоды, а также показывает детализацию чеков.

## Изменения
- Обновлены Core схемы и добавлены новые способы оплаты (`sbp`, `legal_entity_account`).
- Реализован API эндпоинт `GET /api/reports/sales` в Core, отдающий подробную сводку и агрегацию по методам оплаты.
- В UI-модуле добавлена страница `/reports/sales`.
- Страница корзины теперь предлагает выбрать `Счёт юрлица` в выпадающем списке при оформлении.
- В чек добавлен вывод выбранного способа оплаты.

## Использование
Раздел доступен в верхнем меню по кнопке "Отчёты". Выберите один из пресетов периодов ("Сегодня", "Неделя", "Месяц", "Год") или укажите точные даты для получения отчёта.
