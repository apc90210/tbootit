# 08 Avito Module

## Цель

Подготовка товаров к публикации на Авито.

## MVP функции

- список товаров готовых к Авито;
- генерация заголовка;
- генерация описания;
- экспорт CSV/JSON;
- ручные статусы публикации.

## Позже

- API Авито;
- синхронизация статусов;
- автообновление цен;
- снятие проданных товаров.
